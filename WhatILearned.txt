localStorage['duo.state'] remembers all courses when you visit them. It doesn't forget them if you remove them; do that manually. It stores all the courses for a base language when you visit the new site from that language. It *might* be able to pick up courses from other languages if you don't have 0 xp in them. Testing showed that happening, but I don't know if it's consistent.
